# RustDesk Custom Installer

Este repositorio contiene un script de PowerShell y un archivo `.iss` de Inno Setup para instalar RustDesk y aplicar configuraciones.

## Archivos

- `rd-config.ps1`: Script de configuración post-instalación.
- `rustdesk-bundle.iss`: Script de empaquetado de Inno Setup.

## Nota

El objetivo actual es analizar por qué las configuraciones no se aplicaron en un sistema físico con Windows 11, aunque sí funcionaron en una VM con Windows 10. No se han realizado cambios en los scripts.
